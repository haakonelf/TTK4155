CLONE FROM GIT:
git clone https://github.com/haakonelf/TTK4155.git


PUSH TO GIT:
git add *       # for nye filer
git commit -a   # -a er for å adde modifiserte filer
git push	# push to git




STATUS COMMANDS:
git status  # list all modified/new/untracked files, green is added, red is untracked.
git diff    # list differences from last commit




VIM EDITOR COMMANDS:
j = down
k = up
h = left
l = right

x = delete
u = undo
Ctrl+r = redo

i = insert mode
Esc = Exit to navigation mode

Ctrl+v = Visual block mode

:wq = save and exit
